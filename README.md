# Zamkham Ali - Portfolio Website

A modern, responsive portfolio website with smooth animations and dark theme.

## Setup Instructions

1. **Add Your Profile Image**
   - Save your profile picture as `profile.jpg` in the root directory
   - The image should be square or will be cropped to a circle
   - Recommended size: 600x600px or larger

2. **Open the Website**
   - Simply open `index.html` in your web browser
   - Or use a local server for best experience:
     ```bash
     # Using Python
     python -m http.server 8000
     
     # Using Node.js
     npx http-server
     ```
   - Then visit `http://localhost:8000`
